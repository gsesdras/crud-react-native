# crud-react-native
Um sistema CRUD bem simples no intuito de criar portifólio e agregar conhecimento no desenvolvimento de aplicações web e mobile.

## Funcionalidades
- Login com autenticação
- Cadastro
- Sair
- Esqueci minha senha
- Trocar senha pela Dashboard
